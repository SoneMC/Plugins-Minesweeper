# Minesweeper
> Have you ever seen Minesweeper in Minecraft?

## Features
- GUI
- Safe main island
- Flags
- Mines detection
- 6x9
  
## Commands and Permissions

| Command              | Description                | Permission             |
|----------------------|----------------------------|------------------------|
| `/minesweeper`       | Starts game                | `minesweeper.play`     |
